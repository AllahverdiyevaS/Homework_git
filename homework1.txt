 1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

На GitHub.com перейти на главную страницу репозитория.скопировать URL-адрес репозитория.
потом  в  GitBash ввести git clone и вставить URL-адрес, скопированный ранее.

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
git init, git add, git commit,
git remote add origin,git branch -M,git push -u origin main

3.Как получить на локальный компьютер изменения с github.com
git pull

4.Какую информацию нам дают команда :
git status-можем посмотреть текущее состояние репозитория.
git log-показывает хронологическую последовательность всех commit от старого к новому
git remote -v  показывает связан ли наша директория с github.com
